EMS Project Overview

Role: Project Lead and Full Stack Developer

Technologies: Java Spring Boot, Angular, MySQL, AWS/Heroku (Deployment), HTML, CSS, JavaScript

Project Description: Spearheaded the development of a robust ERP system for organizations, streamlining processes across three core modules: RMS (Registration Management System), LMS (Learning Management System), and BDM (Business Development Management).

Key Features:

  • Registration Management (RMS): Simplified and automated employee and student registration processes.
  • Learning Management (LMS): Provided a dynamic platform for tracking learning activities, materials, and assessments.
  • Business Development Management (BDM): Supported end-to-end HR operations, including interview scheduling, employee onboarding, performance tracking, and payroll generation.
  • Integrated Reporting: Enabled comprehensive dashboards and reporting for all modules.
  • Cloud Deployment: Deployed on AWS/Heroku for high availability and scalability.

Key Achievements:

  • Successfully led a cross-functional team to deliver the project on time and within scope.
  • Defined project architecture, established coding standards, and ensured best practices were followed.
  • Mentored team members on technology and design decisions, fostering skill development.
  • Delivered a user-friendly, scalable system that met organizational requirements and received positive feedback.

Responsibilities:

  • Oversaw the entire software development lifecycle, from requirements gathering to deployment.
  • Designed and implemented backend services using Spring Boot for multi-module functionality.
  • Developed Angular-based frontend for seamless interaction across modules.
  • Structured database schema in MySQL for efficient data management and inter-module communication.
  • Ensured application security through robust authentication and role-based access control.
