JobSupport4U Project Overview

Role: Full Stack Developer

Technologies: Java Spring Boot, Angular, MySQL, AWS/Heroku (Deployment), HTML, CSS, JavaScript

Project Description: Developed a comprehensive freelance and employee marketplace platform to connect freelancers with employers. The platform enables freelancers to register, browse available projects, and submit proposals. Employers can post projects, evaluate submissions, and manage hiring. Admin functionalities ensure smooth platform operation, including user management and project moderation.

Key Features:

  • User Roles: Seamlessly managed functionalities for freelancers, employers, and admins.
  • Proposal System: Designed a dynamic submission and evaluation workflow for project proposals.
  • Admin Panel: Integrated admin guidelines for efficient management of users, projects, and payments.
  • Scalability: Deployed the platform on AWS/Heroku, ensuring reliability and scalability.
  • Responsive UI: Designed a responsive, user-friendly interface using Angular and Bootstrap.

Key Achievements:

  • Successfully developed and deployed a scalable platform for freelance and employee interactions.
  • Implemented a secure proposal submission and evaluation workflow.
  • Optimized platform performance for better user engagement and retention.
  • Designed and deployed the platform on AWS/Heroku for improved scalability and uptime.

Responsibilities:

  • Developed backend APIs with Spring Boot for user authentication, proposal management, and project tracking.
  • Built dynamic frontend components using Angular to enhance user experience.
  • Designed and optimized MySQL database schemas for efficient data handling.
  • Implemented payment gateway integration and ensured secure transactions.
  • Deployed the application on AWS/Heroku, optimizing for performance and uptime.
